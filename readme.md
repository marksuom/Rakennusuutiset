## Hyödylliset komennot

### Git

``git status``` — näyttää mitä tiedostoja on muutettu
`git add .` — lisää kaikki muutokset seurantaan
````git commit -m "viesti"` — tallentaa tilanteen historiaan
`git log` — näyttää muutoshistorian
